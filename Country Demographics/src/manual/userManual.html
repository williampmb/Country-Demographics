Basic Functions

Viewing Countries

To view a Country, click on the dropdown menu for Continents and select a Continent. Then, click on the list of Countries and select a Country that belongs to the above Continent.

Searching For Countries

To search for a Country start typing the name of the Country in the Search box. Click the Country name that you are searching as it appears in the dropdown menu.

Change Your Password

To change your password, go to 'Edit' -> 'Edit User'. Select your user name, type your new password in both password boxes and click 'Save'. The next time you log in you will use that password. Click 'Back' to return to the main screen.

Admin Functions

Edit a Continent

To edit a Continent, go to 'Edit' -> 'Edit Continent'. Select a Continent from the dropdown menu, edit its name and click 'Save'. Click 'Back' to return to the main screen.

Create a Continent

To create a Continent, go to 'Edit' -> 'Edit Continent'. Click the 'New' button. A new Continent will appear in the Continent list. The default name is 'New Continent', select it and edit it to change the name.

Delete a Continent

To delete a Continent, go to 'Edit' -> 'Edit Continent'. Select a Continent from the dropdown menu and click the 'delete' button. If the Continent does not contain any Countries, it will be delete. If it does contain at least one Country, it will display an error message, and the Continent will not delete. Click 'Back' to return to the main screen.

Edit a Country

To edit a Country, go to 'Edit' -> 'Edit Country'. Select a Continent from the first dropdown menu. Then, select a Country from the second dropdown menu. Edit any of the text fields. To change the photo, click the 'Browse button and browse for an image file and open it. Click 'Save' to save your changes. Click the 'Back' button to return to the home screen.

Create a Country

To create a Country, go to 'Edit' -> 'Edit Country'. Select a Continent from the first dropdown menu. Then, click the 'New' button. A new Country with the name 'New Country' will be created in the dropdown list of Countries. Click the 'Back' button to return to the home screen.

Delete a Country

To delete a Country, go to 'Edit' -> 'Edit Country'. Select a Continent from the first dropdown menu. Then, select a Country from the second dropdown menu. Click the 'Delete' button, and the Country will be deleted. Click the 'Back' button to return to the home screen.

Edit a User

** As an admin, it is possible to edit any user's password, username and user type. **
To edit a User, go to 'Edit' -> 'Edit User'. Select a User from the dropdown menu. Edit any of the values and and click 'Save'. Any changes will come into affect the next time a user logs in. Click the 'Back' button to return to the home screen.

Create a User

To create a User, go to 'Edit' -> 'Edit User'. Click the 'New' button. A new User will be created with the name 'New User'. You can now select that user in the User dropdown menu.

Delete a User

To delete a User, go to 'Edit' -> 'Edit User'. Click the 'Delete' button. You cannot delete the User currently logged in.
